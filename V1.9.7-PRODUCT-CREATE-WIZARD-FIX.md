# Pool Shed v1.9.7

## Product creation wizard fix

The grouped Product Hub renderer now rebinds the product creation wizard after each screen render.

Fixed behaviours:
- Next step advances through all eight creation stages.
- Previous returns to the prior stage.
- Numbered stage tabs open their matching page.
- Progress bar, stage title and final Create Product action update correctly.
- Product profile tabs are rebound after opening a product.

No database migration is required.
