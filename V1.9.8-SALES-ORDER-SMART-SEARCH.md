# Pool Shed v1.9.8

## Sales order smart catalogue search

- Searches all active catalogue records, including products created by the grouped CSV importer.
- Searches parent name/SKU, variant name/value, Pool Bros SKU, supplier SKU, barcode, brand, category, supplier and description.
- Supports multi-word matching such as `chlorine tablets 5 kg`.
- Displays parent and variant information in the result picker.
- Shows recent active catalogue items when the search field receives focus.
- Uses safe aliases for imported price and supplier fields.
- Excludes archived, deleted and hidden non-stock charge records.
- Adds keyboard Arrow Down navigation and retains Enter-to-add behaviour.
- Refreshes the service-worker cache as v1.9.8.
